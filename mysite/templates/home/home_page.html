What does our web application do?

Linguistic Learners takes the frames of a video containing a persons face and creates facial landmarks from these frames. From there we use temporal convolutional networks to detect the persons mouth moving over time. After we have these frames we can then predict the word being spoken using a machine learning model with the ResNet-34 as the base architecture.
